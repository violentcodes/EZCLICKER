# EZCLICKER
## WHAT IS IT?
A SIMPLE PYTHON PROGRAM WHICH CAN BE USED TO AUTOCLICK /AUTOCLICKER

* mouse
* colorama
* pyfiglet
## how to run and install
* ```open cmd```
* ```pip install colorama```
* ```pip install mouse```
* ```pip install pyfiglet```
* after this u can run  `main.py`
